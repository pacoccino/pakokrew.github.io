Acidoscope

Acidoscope is a web-based interactive animation visualisers.

Using WebGL and Pixel Shaders, it displays real-time trippy visualisations, controllable via MIDI devices. If you use a music production software like Ableton Live, they are synchronised with the tempo via the network !

VueJS - WebGL - WebVR - GLSL - NodeJS

acidoscope.art

Willet

Willet is a web-based multi crypto-currency wallet. Its goal is to be easy-to-use, fast and secure.

Supporting Bitcoin, Ethereum and Stellar Lumens. Based on Stellar blockchain, it is fully-decentralized. Private keys are only holded client-side.

React - Redux - NodeJS - PWA - Stellar

willet.io

Producr

producr is a web application helping to connect directly music producers and consumers by allowing the latter to donate money.

Users can deposit crypto currencies on their account and connect with their soundcloud account. Each time they listen a track, they give a tip to the producer. Producers can then connect and get their money as crypto-currency. Without third-party.

React - Redux - NodeJS - Ethereum - Material
